# tester-pickles2-intergration-test

Pickles 2 のための結合テスト環境です。

## Usage - 使い方

```
cd /path/to/your/directory/;
git clone https://github.com/tomk79/tester-pickles2-intergration-test.git;
cd tester-pickles2-intergration-test/;
composer install;
composer test;
```
